# GCal Toggle All Day
- Toggles Google Calendar events between timed and All Day using *Ctrl+Enter*
- [Click here to install](https://github.com/rawbytz/gcal-toggle-all-day/raw/master/GCalToggleAllDay.user.js) in your script manager. (Tampermonkey or Greasemonkey)
## To Use:
- Click the event once to display the model dialog.
- With the modal present, activate *Ctrl+Enter*
